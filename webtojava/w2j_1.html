前端人员转Java小册子

第一章,选定了就要坚持

从前端转做java,通过查看百度上的一些建议和方法,或许是奏效的,也有可能是将你带向并不是真正”java”的”伪java”,但是既然想转到java,就要符合一些基本的规则:

  • 1. 网络上的java推荐课,博客等纷繁杂乱,初学者很难找准到底哪个是对的,符合要求规范的.

  • 2. 学习到一定阶段,没有督促,容易懈怠,之后就又做回前端了.

  • 3. 很难有学习速度上的提升,遇到一些棘手的问题容易卡壳,难以为继.

这里,我将针对上述简单三点做出相对的解决方案.也希望你能好好认真学习.

针对第一点,我将会结合MOOC中国大学翁恺老师的课程以及部分较好的培训机构课程,在线教程等来整理整体的学习路线.

希望在此期间,你能根据所提要求,熟悉java概念,完成编程操作,并最好记录博客,用来督促自己.

针对第二点,我将会结合一些学习方法和一些辅助软件提供给你,让你去做一个真正的”学习者”,而不是阅读者.

在上手期间,也要知道自己做的是什么,完成的是什么操作,不能以码为码,随码而码,要知道你码的是什么,什么作用,哪里是正确的,哪里自己还欠缺着的.

针对第三点,我还是结合一些笔记,博客的管理,来进行知识的快速迭代,学过去的暂时可以先忘记,专注当下要做的命题,并要反过头来回顾之前的学习.

第二章,Java的基本概念

闲话不多说,本章从介绍Java的基本概念开始.


Java是这个老头何其团队的结晶,其它的不过多介绍,要看历史的请移步百度.
因为前端有变量的概念和Java内的变量几乎差不多,所以从这里入手还是挺快的.


如上图示是java的基本数据类型,不像使用js时,不需要指定数据类型便可以创建变量,俗话说的”松散型”语法.而Java虽然在java10也会使用var来声明变量,但现在免费版就是java8.

如下所示:
js声明变量需要写在script标签内或js文件中.而java的变量也是要写在”类”中的.

public class TestJava1{



    private String username;

    private Integer password;



}

类以class作为开始,类似js方法的function,而class后面是类名TestJava1,最终这个类也会是叫做TestJava1.java,如果名称不对称,是会出错的.(现在IDE是自动智能的这个不用担心)

跟js不同的,private作为访问控制限定符,只允许本类去访问.其它的访问限定符还有:

public

protected

default/friendly  <其实就是不写

private

这个访问限定符跟var和let的作用类似.具体控制范围自行百度.

依旧是看这个类,String是字符串,而Integer是属于int类型的包装类.

我们一般喜欢使用包装类,因为包装类能解决原始类型的一些不好的地方.

如: 在变量里需要保存用户的性别

public class TestJava2 {



    private static int sex;

    private static Integer sex_;



    public static void main(String[] args) {



        System.out.println(sex);  //输出0

        System.out.println(sex_);  //输出null



    }

}

如果我们指定0为女性,1为男性,那么使用int时会默认时0,这就容易发生混淆.
Integer的好处在于它可以声明为null.

接下来来实现一个最基本的加减法.

    //本类的私有方法,add

    private static Object add(Object a, Object b) {

        //转换为double类型

        double v = Double.parseDouble(a.toString());

        double v1 = Double.parseDouble(b.toString());

        return v + v1;

    }

上下面方法进行了类型转换,请注意

    //本类的私有方法,minus

    private static Object minus(Object left, Object right) {

        //转换为double类型

        double lv = Double.parseDouble(left.toString());

        double rv = Double.parseDouble(right.toString());

        return lv - rv;

    }

Java的函数最先接触到的肯定是main,没有main函数很难在控制台(console)打印出hello world
main函数的写法如下:

    public static void main(String[] args) {



       //code



    }

在main函数中,可以使用如下语句进行控制台的语句打印输出:

System.out.println("hello world!");

继而编译该类,执行编译后代码就可以在cmd的黑窗口中看到语句的输出了.
而我们这里讲的都是在编辑器(IDEA,Eclipse)中的一些操作,所以先忽略黑窗口了.
